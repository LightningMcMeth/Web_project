Information:

  • Start the server by going to the project directory root (finalProject) and entering the following command: daphne -p 8000 finalProject.asgi:application
  • socket endpoint (http://127.0.0.1:8000/ws/):
    This endpoint showcases socket functionality. User can enter any text and send it to the server.
    The entered text will be displayed on the page along with the server response.
    The server will respond with a "pong" only if the user sends "ping".
  • Information endpoint (http://127.0.0.1:8000/info/)
    This endpoint is accessed via the example url. You're already here.
